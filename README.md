# FoodReceipt

카탈로그 엑셀과 발주서 엑셀을 비교해 거래명세서를 자동 생성하는 도구입니다.

## 주요 흐름

1. 발주서 파일을 선택합니다.
2. 카탈로그 파일을 선택합니다.
3. 처리할 날짜 시트를 선택합니다.
4. 출력 폴더를 확인하고 거래명세서를 생성합니다.

생성 결과에는 섹션별 거래명세서와 `매칭_검토리포트_YYYY-MM-DD.xlsx`가 포함됩니다.

## 개선된 처리 방식

- 발주서와 카탈로그를 고정 행/열이 아니라 헤더 기반으로 탐지합니다.
- `식품명`, `품목명`, `제품명`, `상품명`, `품명` 등의 제품명 헤더를 함께 인식합니다.
- `규격`, `용량`, `중량`, `포장단위` 등의 규격/용량 헤더를 함께 인식합니다.
- `단가`, `가격`, `공급가`, `납품단가` 등의 가격 헤더를 함께 인식합니다.
- `1kg`과 `1000g`, `1L`와 `1000ml`, `500g x 2`와 `1kg`처럼 환산 가능한 용량을 비교합니다.
- 매칭 결과를 `matched`, `review`, `unmatched`로 분리하고 신뢰도와 사유를 리포트에 기록합니다.
